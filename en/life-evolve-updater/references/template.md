# Life Evolve Template - English

Create these files in the English folder:

- `final.md`
- `bridge.md`
- `frontier.md`
- `donelist.md`
- `memory.md`
- `soul.md`
- `hobbies.md`
- `progress.md`

Every file should include `Last updated: YYYY-MM-DD`.

`donelist.md` must include a completion table.

`bridge.md` must connect long-term theses from `final.md` to existing assets in `donelist.md`, near-term validation actions, acceptance criteria, and failure signals.

`progress.md` must distinguish repository completion from project completion.
