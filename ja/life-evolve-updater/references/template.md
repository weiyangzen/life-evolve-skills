# Life Evolve Template - 日本語

日本語フォルダには次のファイルを作る:

- `final.md`
- `bridge.md`
- `frontier.md`
- `donelist.md`
- `memory.md`
- `soul.md`
- `hobbies.md`
- `progress.md`

各ファイルに `最終更新: YYYY-MM-DD` を入れる。

`donelist.md` には completion table を入れる。

`bridge.md` は `final.md` の長期判断を、`donelist.md` の既存資産、直近の検証アクション、acceptance criteria、failure signals に接続する。

`progress.md` では repository completion と project completion を分ける。
