# Life Evolve Skills

Reusable skills for maintaining a living personal strategy repository.

Current skill:

- `life-evolve-updater/`: update a person's `final.md`, `frontier.md`, `donelist.md`, `memory.md`, `soul.md`, and `hobbies.md` from evidence.

This repository is public by design. Do not commit private life histories, raw chats, credentials, or private repository scans here.
